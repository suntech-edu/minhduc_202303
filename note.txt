1. Thẻ để nhúng css và thẻ nhúng javascript
    <link rel="stylesheet" href="">
        -rel: Qui định kiểu file sẽ nhúng vào trang web
            stylesheet là kiểu css
            type="text/css": Nó sẽ hiểu là chúng ta nhúng file css
            href: Đường dẫn của file css

    <script></script>
        - type="text/javascript": quy định file nhúng vào hay code trong thẻ script là javascript
        - src: Đường dẫn của file javascript
